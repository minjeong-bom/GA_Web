# GoodAfternoon App & Web Site

- Quasar.js로 개발되었습니다.
- 다양한 커뮤니티 기능을 지원합니다.
- 편리한 글 작성 기능을 지원합니다.
- 이력서 작성 기능을 지원합니다.

## 디펜던시 설치
```bash
yarn
# or
npm install
```

### 개발 시작하기
```bash
quasar dev
```


### 개발용으로 빌드하기
```bash
quasar build
```

### Customize the configuration
See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).
