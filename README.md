# GoodAfternoon

https://www.goodafternoon.life/

- Quasar.js로 개발되었습니다.
- 다양한 커뮤니티 기능을 지원합니다.
- 편리한 글 작성 기능을 지원합니다.
- 이력서 작성 기능을 지원합니다.

## Dev & Build
### 1. 라이브러리 다운로드
```bash
yarn
# or
npm install
```

### 2. 개발 시작하기
```bash
quasar dev
```


### 3. 개발용으로 빌드하기
```bash
quasar build
```

### Customize the configuration
See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).
